# Contributing to Recollect

Thank you for your interest in contributing! Recollect is a community-driven project, and all contributions are welcome.

## 🤝 How to Contribute
### 1. Report Bugs or Feature Requests
Open an issue in the Issues tab

Use the bug report or feature request template

Include steps to reproduce and expected behavior

### 2. Submit Pull Requests
1. Fork the repository
2. Create feature branch: `git checkout -b feature/amazing-feature`
3. Commit changes: `git commit -m 'Add amazing feature'`
4. Push to branch: `git push origin feature/amazing-feature`
5. Open Pull Request

### 3. Development Setup
```
# Clone your fork

git clone https://github.com/sarox-dev/Recollect.git
cd recollect
```

# Install dependencies
```
pip install -r requirements.txt
```

# Run dev server
```
docker-compose up
```

# 🛠 Development Guidelines
## Commit Messages
```
feat: add local file indexing
fix: resolve search history pagination
docs: update installation guide
style: format UI components
refactor: extract search service
test: add unit tests for indexing
```

# Code Style
Python: Black + isort

JS/TS: Prettier + ESLint

Follow existing code patterns

# Testing
## Write tests for new features

Run tests: `pytest`
